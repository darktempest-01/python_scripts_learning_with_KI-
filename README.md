## 📝 Was ich gelernt habe

### If/Else-Logik

Die richtige Reihenfolge von Bedingungen ist wichtig:

```python
if alter >= 18:      # Zuerst die größte Bedingung
    return "Erwachsen"
elif alter >= 14:    # Dann mittlere Bedingungen
    return "Jugendlicher"
elif alter >= 6:     # Dann kleinere Bedingungen
    return "Kind"
else:                # Zum Schluss der Rest
    return "Kleinkind"
```

### Funktionen mit return

- `return` gibt einen Wert zurück
- Ohne `return` gibt eine Funktion `None` zurück
- `print()` in einer Funktion gibt nur aus, aber gibt nichts zurück
